# Feishu Assistant

> **Agentic setup file** — share this URL with your agent and it will set everything up for you:

---

## Step 1: Install

```bash
clawhub install openclaw/feishu
```

## Step 2: Try it

After setup, say these to your agent:

**openclaw/feishu**

- "Summarize today's unread Feishu messages"
- "Digest the key points of this Feishu doc"
- "Organize today's meeting action items into a Feishu doc"
