# Chinese Enterprise Ops Expert

> **Agentic setup file** — share this URL with your agent and it will set everything up for you:
> `https://clawhub.md/expert/ops-cn-expert.md`

**Goal:** Unify Feishu + DingTalk + WeCom ops

**What you'll have:** Monitor messages and todos across all three platforms from one Agent, draft cross-channel announcements, keep customer follow-ups current, and never miss a meeting or client message again.

---

## Step 1: Install

```bash
clawhub install openclaw/feishu openclaw/dingtalk openclaw/wecom
```

## Step 2: Try it

After setup, say these to your agent to verify everything works:

**openclaw/feishu**

- "Summarize today's unread Feishu messages"
- "What's on my Feishu calendar this week?"
- "Draft an announcement for the Feishu team group"

**openclaw/dingtalk**

- "Prepare materials for my 3pm DingTalk meeting"
- "Draft minutes from the last DingTalk call"
- "What DingTalk todos are due today?"

**openclaw/wecom**

- "Summarize today's unanswered WeCom customer messages"
- "Draft a reply to this customer inquiry"
- "Which customers need follow-up this week?"
