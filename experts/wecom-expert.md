# WeCom Customer Service Assistant

> **Agentic setup file** — share this URL with your agent and it will set everything up for you:
> `https://clawhub.md/expert/wecom-expert.md`

**Goal:** WeCom customer follow-up

**What you'll have:** Auto-summarize pending customer messages, generate reply drafts, and mark follow-up status — so your response speed improves by an order of magnitude.

---

## Step 1: Install

```bash
clawhub install openclaw/wecom
```

## Step 2: Try it

After setup, say these to your agent to verify everything works:

**openclaw/wecom**

- "Summarize all unanswered customer messages from today"
- "Draft a reply to this inquiry message"
- "List the customers I need to follow up with this week"
